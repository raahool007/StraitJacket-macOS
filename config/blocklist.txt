# DNS-over-HTTPS (DoH) resolver endpoints
# ------------------------------------------------------------
# Browsers with "Secure DNS"/DoH on resolve names themselves over HTTPS,
# bypassing the local sinkhole. Sinkholing the DoH SERVER hostnames makes those
# lookups fail, so most browsers (in default "automatic" mode) fall back to the
# system resolver and are filtered again. PARTIAL mitigation only: a browser set
# to "strict" DoH, a hardcoded resolver IP, or a VPN still bypasses this. The
# reliable fix is to disable Secure DNS in the child's browser.
# ============================================================
